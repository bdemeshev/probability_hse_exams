\(H_0\) верна, равна
\begin{answerlist}
  \item \(\frac{\sum x_i^2 - 4\sum x_i+2}{n}\)
  \item \(\frac{\sum x_i^2 - 4\sum x_i}{n}\)
  \item \(\frac{\sum x_i^2 - 4\sum x_i}{n}+4\)
  \item \(\frac{\sum x_i^2 - 4\sum x_i}{n}+2\)
  \item \(\frac{\sum x_i^2 - 4\sum x_i+4}{n}\)
\end{answerlist}
\end{question}

  \item Неверно
